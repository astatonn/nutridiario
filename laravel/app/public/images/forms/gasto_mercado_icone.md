# Prompt: Gasto com Mercado (ÍCONE)

[CONCEITO & TEMA]
- Tema principal: Orçamento alimentar/Gastos com supermercado
- Mensagem transmitida: Planejamento financeiro, investimento em alimentação, gestão de budget
- Abordagem: Design de ícones monetários sem personagens

[COMPOSIÇÃO VISUAL PRINCIPAL]
- Elemento central: Carrinho de compras + símbolo de dinheiro (R$) ou moedas
- Estilo: Ícones financeiros minimalistas
- Tamanho: Ícones em escala progressiva (menos $ a mais $$$)
- Posição: Centro com elementos ao redor

[ELEMENTOS ICONOGRÁFICOS]

1. **Carrinho de compras** - Ícone de carrinho de supermercado estilizado
2. **Símbolo monetário** - R$ ou cifrão em destaque
3. **Moedas/notas** - Representação estilizada de dinheiro
4. **Sacolas de compras** - Ícones de sacolas com alimentos
5. **Gráfico de barras** - Pequeno gráfico indicando níveis de gasto (baixo/médio/alto)

[COMPOSIÇÃO DE FUNDO]
- Shape de fundo: Orgânico/Blob/Liquid shape semi-irregular
- Forma: Ondulada com bordas suaves
- Cor do shape: Verde economia (#81C784)
- Tamanho do shape: 65% da composição
- Posição: Centralizado

[ELEMENTOS DECORATIVOS]
- Símbolos de economia: Porquinho, cofre estilizado
- Etiquetas de preço: Tags pequenos com valores simbólicos
- Setas de crescimento: Indicando diferentes níveis de gasto
- Percentuais: Símbolos de % para budget

[HIERARQUIA VISUAL]
1. Carrinho + símbolo R$ (destaque máximo)
2. Moedas/notas (secundário)
3. Gráfico e etiquetas (terciário)
4. Shape de fundo (suporte)

[PALETA DE CORES COMPLETA]
- Cor primária: #4CAF50 (verde economia)
- Secundária: #81C784 (verde claro)
- Acentos: #66BB6A (verde médio)
- Dinheiro: #FFC107 (dourado/amarelo)
- Ícones: #424242 (cinza escuro)
- Shape fundo: #C8E6C9 (verde suave)
- Cores de apoio: #FFFFFF (branco), #000000 (preto)

[ESPECIFICAÇÕES TÉCNICAS OBRIGATÓRIAS]
✓ Fundo transparente (PNG com alpha channel)
✓ Apenas ícones visíveis (SEM PERSONAGENS)
✓ Shape com formato blob/líquido irregular suave
✓ Contornos pretos 2-3px uniformes
✓ Flat design 2D vetorial
✓ Sem texturas ou gradientes complexos
✓ Estilo minimalista financeiro
✓ Símbolos monetários reconhecíveis
✓ Resolução mínima: 1500x1500px
✓ Espaço respiratório nas bordas
✓ Composição clara e organizada
✓ Shape orgânico seguindo tendências de design moderno
