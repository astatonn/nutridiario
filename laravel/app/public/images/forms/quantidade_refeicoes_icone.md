# Prompt: Quantidade de Refeições (ÍCONE)

[CONCEITO & TEMA]
- Tema principal: Frequência alimentar/Distribuição de refeições ao longo do dia
- Mensagem transmitida: Organização das refeições, ritmo metabólico, distribuição adequada de energia
- Abordagem: Design de ícones minimalista sem personagens

[COMPOSIÇÃO VISUAL PRINCIPAL]
- Elemento central: 5 ícones de pratos simples dispostos em linha horizontal ou semicírculo
- Estilo dos pratos: Círculos clean com símbolo minimalista de talheres (garfo + faca)
- Cores dos pratos: Gradação de laranja (#FF9800 → #FFB74D → #FFCC80 → #FFE0B2 → #FFF3E0)
- Tamanho: Pratos com tamanhos ligeiramente diferentes para criar dinamismo
- Posição: Centro da composição

[ELEMENTOS ICONOGRÁFICOS]
- Ícones de relógio: Pequenos relógios minimalistas ao lado de cada prato
- Horários representados: 7h, 10h, 13h, 16h, 19h (em formato visual clean)
- Números: Numeração sutil (1, 2, 3, 4, 5) abaixo de cada prato
- Linhas conectoras: Linhas pontilhadas sutis conectando os pratos em sequência
- Setas de fluxo: Pequenas setas indicando progressão temporal

[COMPOSIÇÃO DE FUNDO]
- Shape de fundo: Orgânico/Blob/Liquid shape semi-irregular
- Forma: Circular suave com bordas onduladas
- Cor do shape: Laranja claro (#FFE0B2)
- Tamanho do shape: 65-70% da composição
- Posição: Centralizado atrás dos ícones
- Transparência do shape: 70% de opacidade

[ELEMENTOS DECORATIVOS]
- Símbolos de sol: Ícone de sol para refeições matinais
- Símbolos de lua: Ícone de lua para refeição noturna
- Ícones de energia: Pequenos raios ou estrelas representando energia distribuída

[HIERARQUIA VISUAL]
1. Ícones de pratos (mais destacados)
2. Relógios mostrando horários (secundário)
3. Linhas conectoras e numeração (terciário)
4. Shape de fundo (suporte)

[PALETA DE CORES COMPLETA]
- Cor primária: #FF9800 (laranja energia)
- Gradação: #FFB74D → #FFCC80 → #FFE0B2
- Cor shape fundo: #FFF3E0 (laranja muito claro)
- Cor ícones: #424242 (cinza escuro)
- Cores de apoio: #FFFFFF (branco), #78716C (cinza médio), #000000 (preto)

[ESPECIFICAÇÕES TÉCNICAS OBRIGATÓRIAS]
✓ Fundo transparente (PNG com alpha channel)
✓ Apenas ícones visíveis (SEM PERSONAGENS)
✓ Shape com formato blob/líquido irregular suave
✓ Contornos pretos 2-3px uniformes nos ícones
✓ Flat design 2D vetorial
✓ Sem texturas ou gradientes complexos
✓ Estilo minimalista e clean
✓ Resolução mínima: 1500x1500px
✓ Espaço respiratório nas bordas
✓ Composição centralizada e balanceada
✓ Shape orgânico seguindo tendências de design moderno (liquid/blob shapes)
✓ Ícones seguindo Material Design ou similar (simplicidade e clareza)
