# Prompt: Restrições Alimentares (ÍCONE)

[CONCEITO & TEMA]
- Tema principal: Limitações dietéticas/Alergias/Intolerâncias
- Mensagem transmitida: Atenção a restrições, cuidados especiais, opções alternativas
- Abordagem: Design de ícones informativos sem personagens

[COMPOSIÇÃO VISUAL PRINCIPAL]
- Elemento central: 4-5 ícones de alimentos com símbolos de alerta/restrição
- Estilo: Ícones de alimentos com círculos vermelhos com barra ou símbolo X
- Tamanho: Ícones médios uniformes
- Posição: Grid horizontal ou circular

[ELEMENTOS ICONOGRÁFICOS - RESTRIÇÕES]

1. **Lactose** - Ícone de leite/queijo com símbolo ⊘ (proibido) | Cor: #2196F3 (azul)
2. **Glúten** - Ícone de pão/trigo com símbolo ⊘ | Cor: #FF9800 (laranja)
3. **Vegetariano** - Ícone de carne com símbolo ⊘ + vegetais verdes ✓ | Cor: #4CAF50 (verde)
4. **Vegano** - Ícone de produtos animais com ⊘ + plantas ✓ | Cor: #66BB6A (verde claro)
5. **Outros** - Ícone de interrogação com prato | Cor: #9E9E9E (cinza)

[COMPOSIÇÃO DE FUNDO]
- Shape de fundo: Orgânico/Blob/Liquid shape semi-irregular
- Forma: Suave ondulada
- Cor do shape: Bege claro (#F0EAD6)
- Tamanho do shape: 65% da composição
- Posição: Centralizado

[ELEMENTOS DECORATIVOS]
- Círculos de atenção: Bordas circulares ao redor de ícones restritos (vermelho #EF4444)
- Check marks verdes: Para alimentos permitidos (#4CAF50)
- Labels: Nome da restrição em texto pequeno abaixo
- Símbolos médicos: Cruz médica sutil para alergias graves

[HIERARQUIA VISUAL]
1. Ícones de alimentos com restrição (destaque)
2. Símbolos de proibição/permissão (secundário)
3. Labels textuais (terciário)
4. Shape de fundo (suporte)

[PALETA DE CORES COMPLETA]
- Restrição/Alerta: #EF4444 (vermelho)
- Permitido: #4CAF50 (verde)
- Lactose: #2196F3 (azul)
- Glúten: #FF9800 (laranja)
- Vegetal: #66BB6A (verde claro)
- Shape fundo: #F0EAD6 (bege claro)
- Cores de apoio: #FFFFFF (branco), #000000 (preto), #9E9E9E (cinza)

[ESPECIFICAÇÕES TÉCNICAS OBRIGATÓRIAS]
✓ Fundo transparente (PNG com alpha channel)
✓ Apenas ícones visíveis (SEM PERSONAGENS)
✓ Shape com formato blob/líquido irregular suave
✓ Contornos pretos 2-3px uniformes
✓ Flat design 2D vetorial
✓ Sem texturas ou gradientes complexos
✓ Estilo minimalista informativo
✓ Símbolos universalmente reconhecíveis
✓ Resolução mínima: 1500x1500px
✓ Espaço respiratório nas bordas
✓ Composição centralizada e clara
✓ Shape orgânico seguindo tendências de design moderno
